CTI Dark Web Monitor
Objectif

Outil de Cyber Threat Intelligence permettant de détecter des fuites de données et mentions d'entités sur le Dark Web (.onion).
Fonctionnalités prévues

    Crawler Tor (.onion)
    Scraping de données OSINT
    Détection d'entités sensibles
    Analyse NLP simple
    Base de connaissance Obsidian
    Dashboard Streamlit
    Système RAG (LlamaIndex)
    CI/CD avec GitHub Actions + Docker
